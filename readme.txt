Unzip zip file and run with Arduino code from mBlock.

controls:
	V change camera view 
		-follow car
		-on top of car
		-free camera
	
	in follow car mode
		scroll +/-	zoom in/out
	
	in free camera mode:
		I, J, K, L 	move camera 
		U, P 		lower/raise camera
		mouse		look
		ESC			quit
		RETURN		start executing code
		
		
		
For development, unzip deps and move the deps folder outside of project folder, alongside it.
